# Microservices Medical System

## Deskripsi
Sistem ini menggunakan arsitektur microservices dengan Node.js, gRPC, RabbitMQ, dan Docker.

## Teknologi
- Node.js
- Express
- gRPC
- RabbitMQ
- Docker
- HTML + Bootstrap

## Fitur
- Registrasi pasien
- Message Queue (RabbitMQ)
- Komunikasi gRPC
- Dashboard frontend

## Cara Menjalankan
```bash
docker compose up --build
```
